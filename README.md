Простое веб-приложение для просмотра текущей погоды в любом городе мира!
Особенности:
- Поиск погоды по названию города
- Отображение температуры, влажности, давления и ветра
- Красивый дизайн с котиком
- Поддержка русского языка
Как использовать:
1. Введите название города (на русском или английском)
2. Нажмите кнопку "Тык" или клавишу Enter
3. Получите актуальную погоду
Как получить API-ключ:
1. Перейдите на сайт OpenWeather (https://openweathermap.org/) и зарегистрирйтесь.
2. Перейдите в раздел My API Keys и скопируйте его.
3. Откройте файл "script.js" в проекте, найдите строку: "const API_KEY = 'YOUR_API_KEYS', и вставьте свой API ключ.
Важно:
- Активация API ключа может занять более 10 минут.
- Бесплатный тариф включает 1000 запросов в день.
- Никому не передавайте свой API ключ!
